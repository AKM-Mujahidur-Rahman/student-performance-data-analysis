# Student Performance Data Analysis

## Overview
This project analyzes student exam performance and the factors that may affect exam scores. It uses statistical analysis, visualization, and basic regression modeling to understand student performance patterns.

## Dataset
The dataset contains student performance-related features such as study hours, attendance, sleep hours, previous scores, tutoring sessions, motivation level, internet access, family income, teacher quality, school type, physical activity, and exam score.

## Objectives
- Understand the structure of the dataset
- Analyze numerical and categorical variables
- Explore relationships between student habits and exam scores
- Identify key factors that influence academic performance
- Apply basic linear regression for prediction/analysis
